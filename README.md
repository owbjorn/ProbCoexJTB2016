# ProbCoexJTB
Some FORTRAN code (that was eventually never used) for the project underlying the 2016 paper [Estimating the probability of coexistence in cross-feeding communities](https://doi.org/10.1016/j.jtbi.2016.07.043)

## Description
The FORTRAN code is intended to simulate the development of a three-species cross-feeding community, described by the replicator equation.

## Usage
The code is in principle run by compilation (_gfortran_ or intel _ifort_ if available) and then running in the terminal, but is added as an archive.
